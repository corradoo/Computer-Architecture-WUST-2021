# Course Overview

This repository contains a collection of solutions for a programming course that covers fundamental concepts in Bash scripting and C programming. The exercises are designed to provide hands-on experience with system-level programming, process management, and low-level I/O operations.

## Bash Solutions

| File | Description |
|---|---|
| [curlAPI.sh](bash/curlAPI.sh) | A script that interacts with web APIs to fetch and display a cat image and a random joke. |
| [hardwareMonitor.sh](bash/hardwareMonitor.sh) | A system monitoring tool that displays PC uptime, network usage, and CPU load. |
| [lcase.sh](bash/lcase.sh) | A utility script that converts all filenames in a specified directory to lowercase. |
| [linuxPS.sh](bash/linuxPS.sh) | A process listing script that shows the PID, PPID, and the number of opened file descriptors for each process. |
| [pong.sh](bash/pong.sh) | A simple implementation of the classic Pong game in Bash. |

## C Solutions

| File | Description |
|---|---|
| [colorHello.c](c/colorHello.c) | A simple program that prints "Hello world" in 256 different colors to the console. |
| [printf&scanf.c](c/printf&scanf.c) | A custom implementation of the `printf` and `scanf` functions using low-level system calls. |
| [rootConsole.c](c/rootConsole.c) | A program that launches a Bash shell with root privileges. |
